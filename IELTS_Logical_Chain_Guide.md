# 📝 IELTS Writing: Logical Chain 构建指南

> **主题**: 如何展开 Extension（论证延伸）  
> **适用**: Task 2 Body 段落  
> **日期**: 2026-03-03

---

## 🔗 什么是 Logical Chain？

```
论点 → 解释 → 结果/例子 → 回扣
```

**核心**: 回答 "为什么？" 和 "怎么样？"

---

## 📊 完整结构

| 部分 | 功能 | 关键词 |
|------|------|--------|
| **Topic Sentence** | 提出观点 | Firstly, Furthermore... |
| **Explanation** | 解释为什么 | This is because, Due to... |
| **Result/Example** | 说明结果/举例 | As a result, For instance... |
| **Link Back** | 回扣论点 | Therefore, This shows that... |

---

## 🎯 3 个万能 Extension 方法

### 方法 1: "因为 - 所以"链条

```
论点 → 因为 A → 因为 B → 所以 C

示例:
Art inspires creativity. 
→ This is because artistic thinking encourages imagination. 
→ Such imagination becomes the foundation for innovation. 
→ Therefore, art drives scientific progress.
```

**适用**: 需要深度论证时

---

### 方法 2: "具体化"法

```
抽象概念 → 具体例子

示例:
Art preserves culture. 
→ Traditional music, paintings, literature carry values. 
→ Chinese calligraphy embodies philosophical thought. 
→ Without artists, cultural identity would be lost.
```

**适用**: 话题太抽象时

---

### 方法 3: "对比 + 例子"法

```
X vs Y → 为什么都需要 → 例子

示例:
Art and science are complementary. 
→ Science answers "how", art addresses "why". 
→ Smartphones need engineering (science) + design (art). 
→ Both are essential for success.
```

**适用**: 比较类题目

## 📝 题 8 完整示范

### Body 1: 为什么重视艺术家

```
【Topic】
People continue to value artists because art addresses fundamental 
human needs that technology cannot satisfy.

【Explanation】
While science provides material comfort, it cannot answer questions 
about meaning, purpose, and beauty. Artists help people process 
emotions and connect with others on a deeper level.

【Example】
During the COVID-19 pandemic, people streamed concerts, read literature, 
and created art to cope with isolation—not technology alone.

【Link Back】
This reveals that art is not a luxury but a fundamental necessity, 
explaining why artists remain valued.
```

**字数**: ~120 字

---

### Body 2: 艺术和科学同等重要

```
【Topic】
Arts and sciences are equally important because they fulfill 
complementary roles in civilization.

【Contrast】
Science focuses on understanding the natural world, while art 
explores and expresses the human experience.

【Example】
Smartphones: engineers (science) create hardware, designers (art) 
make it intuitive and beautiful. Both are essential.

【Link Back】
This interdependence demonstrates that neither is superior—both 
are vital for human progress.
```

**字数**: ~110 字

## 📋 万能模板

### 模板 1: 因果链条

```
【Topic】X is important because...

【Why】This is due to the fact that...

【How】Specifically, ...

【Result】As a consequence, ...

【Example】A clear example is...

【Link】Therefore, ...
```

---

### 模板 2: 对比展开

```
【Topic】X and Y serve different purposes.

【Contrast】While X focuses on..., Y concentrates on...

【Why both】Both are necessary because...

【Example】Consider...

【Link】This shows that...
```

---

### 模板 3: 让步反驳

```
【Concession】Some may argue that...

【Partial truth】It is true that...

【However】However, this overlooks...

【Why】The reason is...

【Example】For instance...

【Link】Thus, the original view is limited.
```

## ✅ Extension 检查清单

写完问自己：

| 问题 | 是/否 |
|------|------|
| 有没有解释"为什么"？ | ☐ |
| 有没有说明"怎么样"？ | ☐ |
| 有没有具体例子？ | ☐ |
| 有没有回扣论点？ | ☐ |

**4 个都是"是"** → Logical Chain 完整！✅

---

## ⚠️ 常见错误

| 错误 | 问题 | 修正 |
|------|------|------|
| **太跳跃** | 论点→结论，无解释 | 加 Because/This is because |
| **无例子** | 只有抽象论述 | 加 For example/Consider |
| **不回扣** | 例子后直接结束 | 加 This shows/Therefore |
| **重复** | 换说法重复论点 | 加具体细节/数据/例子 |

---

## 💡 高分连接词

| 功能 | 词汇 |
|------|------|
| **解释** | This is because, Due to, The reason is that |
| **具体化** | Specifically, In particular, More precisely |
| **结果** | As a result, Consequently, Therefore |
| **举例** | For instance, Consider, A clear example is |
| **对比** | While, Whereas, On the other hand |
| **回扣** | This shows, This demonstrates, Thus |

## 🎯 练习建议

### 每天 10 分钟

1. **选 1 个论点**（从预测题库）
2. **用 1 个方法展开**（因果/具体化/对比）
3. **写 80-100 字段落**
4. **对照 Checklist 检查**
