# DSS 技术指标参数优化报告

**测试时间**: 2026-03-25 08:43-08:46
**测试数据**: A股市场6只代表性股票（贵州茅台、工商银行、中国国航、中国平安、五粮液、平安银行）
**数据范围**: 每只股票300天历史数据

---

## 一、优化目标

在真实股票数据上优化DSS技术指标计算参数：
1. Kalman去噪参数优化
2. RSI周期参数优化
3. MACD参数优化
4. 评分权重优化

---

## 二、优化结果汇总

### 2.1 Kalman去噪参数

| 参数 | 默认值 | 优化值 | 准确率变化 |
|------|--------|--------|-----------|
| process_noise (Q) | 0.01 | **0.005** | - |
| measurement_noise (R) | 0.1 | **0.5** | - |
| 准确率 | ~55% | **58.2%** | +3.2% |

**分析**:
- 较小的Q值(0.005)意味着更平滑的状态估计
- 较大的R值(0.5)表示对观测数据的信任度降低，更多依赖模型预测
- 这种组合在A股波动性较大的市场中表现更好

### 2.2 RSI周期参数

| 周期 | 准确率 |
|------|--------|
| 7 | 54.4% |
| **9** | **54.7%** ✓ |
| 14 (默认) | 54.2% |
| 21 | 53.0% |
| 28 | 52.7% |

**分析**:
- 较短周期(9天)比默认14天略好
- RSI对预测准确率影响有限，变化范围仅2%

### 2.3 MACD参数

| 配置 | 快线 | 慢线 | 信号线 | 准确率 |
|------|------|------|--------|--------|
| 默认 | 12 | 26 | 9 | 54.2% |
| **最佳** | **15** | **30** | **9** | **55.9%** ✓ |

**分析**:
- 较长的快线(15)和慢线(30)周期表现最佳
- 延长了平均线的计算周期，减少了噪音干扰
- 信号线保持默认的9天

### 2.4 评分权重

| 权重配置 | RSI | MACD | MA | 准确率 |
|----------|-----|------|-----|--------|
| 默认 | 1.0 | 1.0 | 1.0 | 54.2% |
| **最佳** | **1.0** | **0.5** | **0.5** | **56.7%** ✓ |

**分析**:
- RSI权重保持1.0
- MACD和MA权重降低到0.5
- 表明RSI信号比MACD和MA均线更可靠

## 三、最佳参数配置

```python
# DSS 最佳参数配置
BEST_PARAMS = {
    # Kalman去噪
    'kalman': {
        'process_noise': 0.005,  # Q参数
        'measurement_noise': 0.5,  # R参数
    },
    
    # 技术指标
    'tech_indicators': {
        'rsi_period': 9,         # RSI周期（默认14）

        'macd_slow': 30,         # MACD慢线（默认26）
        'macd_signal': 9,        # MACD信号线（保持默认）

    # 评分权重
    'scoring_weights': {
        'rsi_weight': 1.0,
        'macd_weight': 0.5,
        'ma_weight': 0.5,
        'volume_weight': 1.0,    # 成交量权重（未优化）
    },
}
```

## 五、关键发现

### 5.1 去噪效果显著
- Kalman滤波参数对预测准确率影响最大
- 高测量噪声(R=0.5)配合低过程噪声(Q=0.005)最佳
- 说明A股噪音较大，需要更强的平滑处理

### 5.2 RSI优于MACD
- RSI权重保持1.0，MACD降至0.5效果最佳
- RSI在超买超卖信号上更可靠

### 5.3 MACD长周期更好
- MACD(15,30,9)比默认(12,26,9)准确率提升1.7%
- 较长周期能更好地过滤市场噪音

### 5.4 评分系统调优
- 评分权重调整可提升2.5%准确率
- RSI信号应给予更高权重
