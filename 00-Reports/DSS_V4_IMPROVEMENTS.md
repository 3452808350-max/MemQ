# DSS v4.0 改进说明

## 基于Kimi建议的改进

### 1. 增加基本面数据
- PE (市盈率)
- PB (市净率)
- ROE (净资产收益率)
- 股息率
- 资产负债率

### 2. 增加情绪分数
- 基于RSI、MACD、均线、成交量计算市场情绪

### 3. 模型集成
- LightGBM + XGBoost 集成预测
- 提高预测稳定性

### 4. 风控机制
- 止损 (默认5%)
- 止盈 (默认15%)
- 仓位计算

### 5. 综合评分
- 技术评分 + 情绪评分 + 基本面评分
