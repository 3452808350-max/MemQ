## 二、DSS问题诊断

### 准确率0%的可能原因：

1. **过拟合** - 100天数据太少，模型泛化能力差
2. **技术指标单一** - 仅RSI+MACD+均线，缺乏市场环境判断
3. **无风险校正** - 未考虑参数不确定性
4. **验证周期过短** - 3天验证窗口太频繁

---

## 三、改进方案

### 3.1 技术路线图

```
当前架构: 数据 → 特征 → 模型 → 信号

改进架构: 
数据 → 市场状态检测 → 自适应指标 → LSTM/GBDT集成 → 贝叶斯校正 → 信号
```

### 3.2 优先级实现

| 阶段 | 改进项 | 预期效果 |
|------|--------|----------|
| **Phase 1** | 添加LSTM模块 | 提升时序建模能力 |
| **Phase 1** | 自适应RSI/MACD | 适应不同市场环境 |
| **Phase 2** | 集成学习 (LSTM+LightGBM) | 多模型融合 |
| **Phase 2** | 贝叶斯参数校正 | 风险控制 |
| **Phase 3** | 扩展验证周期 | 更稳定的准确率 |

### 3.3 具体技术点

**自适应RSI**:
```python
# 根据波动率动态调整周期
period = int(14 * (1 + volatility))
```

**自适应MACD**:
```python
# 根据市场状态切换参数
if market_regime == 'trending':
    fast, slow = 8, 17  # 趋势市
else:
    fast, slow = 5, 35  # 震荡市
```

**LSTM集成**:
```python
# 轻量级LSTM预测
model = tf.keras.Sequential([
```
