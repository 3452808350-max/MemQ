## 🧪 测试命令

### 编译测试

```bash
# Debug 编译
cargo build

# Release 编译
cargo build --release

# 检查代码
cargo check

# 运行测试
cargo test
```

### 运行测试

```bash
# 直接运行
./target/release/dsd-player

# 指定音乐目录
./target/release/dsd-player --music-dir ~/Music/DSD

# 调试模式
./target/release/dsd-player -v
```

### 性能测试

```bash
# 内存占用
/usr/bin/time -v ./target/release/dsd-player

# CPU 占用
top -p $(pgrep dsd-player)
```
