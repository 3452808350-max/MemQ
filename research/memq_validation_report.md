## 3. 改进版 v3 核心优化

### 3.1 模糊词惩罚（新增）

检测低信息词汇，降低评分：

```python
vague_words = ['一下', '一些', '可能', '也许', '差不多', '大概', '看了', '搞了']
if vague_count > 0:
    score *= (0.7 ** vague_count)
```

**修复案例**: "数据源配置了一下" → 分数从 1.365 降至 1.092

### 3.2 否定模式检测 v3（优化）

精确区分"纯否定"和"对比知识"：

```python
def _detect_negation_type(text):
    # 对比知识：有替代方案 OR (转折 + 价值比较)
    if has_alternative or (has_contrast and has_comparison):
        return 'contrastive'  # 奖励 x1.1
    # 纯否定：无替代方案
    return 'pure'  # 惩罚 x0.7
```

**修复案例**:
- "不用 Python，改用 Go" → 对比知识 ✅
- "但不是用于 DSS 项目" → 纯否定 ✅

### 3.3 长度阈值调整

```python
# 原版：<20 字惩罚
if len(text) < 20:
    score *= 0.7

# 改进 v3：<30 字惩罚（更严格）
if len(text) < 30:
    score *= 0.8
```
