# Kimi 协作 Skill

用于与 Kimi 大模型进行金融分析、论文讨论、代码开发协作。

## 快速开始

```python
from skills.kimi_collaboration import kimi_chat

# 基础对话
result = kimi_chat("今天天气怎么样")

# 讨论论文
result = kimi_discuss_paper("StockMixer")

# 代码审查
result = kimi_code_review("/path/to/code.py")
```

## 交流范式

### 任务前缀

| 前缀 | 用途 | 示例 |
|------|------|------|
| `[F-分析]` | 金融分析 | `[F-分析]这只股票的趋势` |
| `[P-精读]` | 论文讨论 | `[P-精读]StockMixer架构` |
| `[C-开发]` | 代码开发 | `[C-开发]实现一个回测函数` |
| `[R-审查]` | 代码审查 | `[R-审查]DSS模型代码` |
| `[S-总结]` | 总结归纳 | `[S-总结]今天的讨论要点` |

### 标准格式

```
[TASK_TYPE] 任务类型

## 背景
(简要说明上下文)

## 任务
(具体要做什么)

## 要求
(格式、长度、语气等)
```

### 示例

**论文讨论**
```
[P-精读] 讨论 StockMixer 论文

## 背景
StockMixer是上海交大团队提出的MLP架构股票预测模型

## 任务
1. 总结核心创新点
2. 分析架构设计
3. 评估实用性

## 要求
中文回复，详细但简洁
```

**代码开发**
```
[C-开发] 实现一个简单的回测函数

## 背景
DSS系统需要回测模块

## 任务
用Python实现：
- 输入：历史价格 + 信号
- 输出：收益率统计

## 要求
- 简洁高效
- 包含文档字符串
```
