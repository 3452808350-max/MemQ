```python
# 二分类 - 涨跌预测
y_binary: 0=下跌, 1=上涨

# 三分类 - 趋势预测
y_trend: 0=下跌(<-10%), 1=震荡(-10%~10%), 2=上涨(>10%)

# 回归 - 涨跌幅预测
y_regression: 下年指数涨跌幅(%)
```

## 🔧 快速开始

```python
import pandas as pd
from sklearn.ensemble import RandomForestClassifier

# 1. 加载数据
X = pd.read_csv('SSE_DSS特征矩阵.csv')
y = pd.read_csv('SSE_DSS目标_二分类.csv')

# 2. 移除年份列
X = X.drop('统计年份', axis=1)

# 3. 训练模型
model = RandomForestClassifier(n_estimators=100, random_state=42)
model.fit(X, y)

# 4. 预测
prediction = model.predict(X.iloc[-1:].values)  # 预测2025年
print(f"2025年预测: {'上涨' if prediction[0] == 1 else '下跌'}")
```

## ⚠️ 注意事项

1. **样本量较小** (n=16)，建议使用交叉验证
2. **时间序列数据**，注意数据泄露问题
3. **标准化处理**，对于距离敏感模型必须标准化
4. **特征选择**，建议根据模型类型选择相关特征
