# 上海证券交易所DSS模型训练数据集 - 数据字典

## 一、数据集概述

| 项目 | 说明 |
|------|------|
| 数据来源 | 上海证券交易所统计年鉴（2005-2025卷） |
| 数据时间范围 | 2004年 - 2024年 |
| 训练样本范围 | 2008年 - 2023年（去除滞后特征缺失值） |
| 总样本数 | 21 |
| 训练样本数 | 16 |
| 总特征数 | 61 |
| 目标变量数 | 5 |

## 二、原始特征（14个）

| 特征名 | 单位 | 说明 |
|--------|------|------|
| 统计年份 | 年 | 数据所属年份 |
| 交易天数 | 天 | 当年实际交易日数量 |
| 上市公司总数 | 家 | 在沪市挂牌交易的公司数量 |
| 新上市公司数 | 家 | 当年新挂牌的公司数量 |
| 上市证券总数 | 只 | 包括股票、基金、债券等各类证券 |
| 股票发行数量 | 亿股 | 总股本（A股+B股） |
| 股票流通数量 | 亿股 | 可流通股本 |
| 股票市价总值 | 亿元 | 按市价计算的总市值 |
| 股票流通市值 | 亿元 | 可流通部分市值 |
| 股票成交金额 | 亿元 | 全年股票交易金额 |
| 集资总额 | 亿元 | 当年股票市场筹资额 |
| 平均市盈率 | 倍 | A股整体市盈率 |
| 上证综合指数 | 点 | 年末收盘点位 |
| 投资者户数 | 万户 | 开户总数 |

## 三、衍生特征（42个）

### 3.1 比率特征（7个）

| 特征名 | 计算公式 | 说明 |
|--------|----------|------|
| 流通市值占比 | 流通市值 / 市价总值 | 反映市场流通性 |
| 流通股本占比 | 流通股本 / 总股本 | 反映股份流通比例 |
| 日均成交金额 | 成交金额 / 交易天数 | 日均交易活跃度 |
| 户均市值 | 市价总值 / 投资者户数 | 人均持仓市值 |
| 证券化率 | 市价总值 / GDP基准 | 股市相对经济规模 |
| 成交额市值比 | 成交金额 / 市价总值 | 换手率估算 |
| 换手率估算 | 成交额市值比 × 100 | 百分比形式换手率 |

### 3.2 同比增长率特征（5个）

| 特征名 | 说明 |
|--------|------|
| 上市公司总数_YoY | 上市公司总数年度变化率 |
| 股票市价总值_YoY | 市值年度变化率 |
| 股票成交金额_YoY | 成交额年度变化率 |
| 投资者户数_YoY | 投资者数量年度变化率 |
| 上证综合指数_YoY | 指数年度变化率 |

### 3.3 滞后特征（12个）

| 特征名 | 说明 |
|--------|------|
| 上证综合指数_lag1/2/3 | 前1/2/3年指数 |
| 平均市盈率_lag1/2/3 | 前1/2/3年市盈率 |
| 股票成交金额_lag1/2/3 | 前1/2/3年成交额 |
| 股票市价总值_lag1/2/3 | 前1/2/3年市值 |

### 3.4 移动平均特征（12个）

| 特征名 | 说明 |
|--------|------|
| 上证综合指数_MA3/MA5 | 3/5年移动平均 |
| 上证综合指数_MA3/MA5_ratio | 当前值与移动平均比率 |
| 平均市盈率_MA3/MA5 | 3/5年移动平均 |
| 平均市盈率_MA3/MA5_ratio | 当前值与移动平均比率 |
| 股票成交金额_MA3/MA5 | 3/5年移动平均 |
| 股票成交金额_MA3/MA5_ratio | 当前值与移动平均比率 |

### 3.5 波动率特征（2个）

| 特征名 | 说明 |
|--------|------|
| 指数波动率 | 3年指数标准差 |
| 市盈率波动率 | 3年市盈率标准差 |

### 3.6 市场状态特征（4个）

| 特征名 | 说明 |
|--------|------|
| 指数年涨跌 | 当年指数涨跌点数 |
| 指数年涨跌幅 | 当年指数涨跌幅(%) |
| 市场状态 | 牛熊分类（强牛市/牛市/震荡/熊市/强熊市）|
| 市盈率历史分位 | 市盈率在历史中的分位点(%) |

## 四、目标变量（5个）

| 目标变量 | 类型 | 说明 |
|----------|------|------|
| 下年指数涨跌 | 回归 | 下一年指数涨跌点数 |
| 下年指数涨跌幅 | 回归 | 下一年指数涨跌幅(%) |
| 下年是否上涨 | 二分类 | 0=下跌, 1=上涨 |
| 下年趋势 | 三分类 | 0=下跌(<-10%), 1=震荡(-10%~10%), 2=上涨(>10%) |
| 下年市场状态 | 文本 | 下一年市场状态描述 |

## 五、使用建议

### 5.1 模型选择

| 任务类型 | 推荐模型 | 目标变量 |
|----------|----------|----------|
| 二分类 | 随机森林、XGBoost、逻辑回归 | 下年是否上涨 |
| 三分类 | 随机森林、XGBoost | 下年趋势 |
| 回归 | 随机森林、XGBoost、线性回归 | 下年指数涨跌幅 |

### 5.2 特征选择建议

| 模型类型 | 推荐特征组合 |
|----------|--------------|
| 基础模型 | 原始特征（14个） |
| 时间序列模型 | 原始特征 + 滞后特征 |
| 技术分析模型 | 原始特征 + 移动平均 + 波动率 |
| 综合模型 | 全部特征 |

### 5.3 数据预处理

1. **缺失值处理**：训练数据已去除含缺失值的样本
2. **标准化**：建议使用StandardScaler进行标准化
3. **特征缩放**：对于基于距离的模型（如KNN、SVM）必须进行缩放
