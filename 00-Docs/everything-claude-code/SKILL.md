# Everything Claude Code

Claude Code 配置大全 - 来自 Anthropic Hackathon 冠军

## 包含内容

### Agents (5个)
- `planner.md` - 功能实现规划
- `architect.md` - 系统架构设计
- `code-reviewer.md` - 代码审查
- `tdd-guide.md` - TDD 开发指南
- `security-reviewer.md` - 安全审查

### Skills (8个)
- `python-patterns.md` - Python 最佳实践
- `tdd-workflow.md` - TDD 工作流
- `security-review.md` - 安全检查清单
- `verification-loop.md` - 验证循环
- `continuous-learning.md` - 持续学习
- `iterative-retrieval.md` - 迭代检索
- `eval-harness.md` - 评估工具

## 使用方法

### 调用 Agent

在对话中可以使用这些子代理：
- `@planner` - 规划功能实现
- `@architect` - 设计系统架构
- `@code-reviewer` - 审查代码
- `@tdd-guide` - TDD 开发指导
- `@security-reviewer` - 安全审查

### 使用 Skill

直接提及技能名称即可激活：
- 讨论 Python 编码规范
- 使用 TDD 工作流
- 进行安全审查
- 设置验证循环
